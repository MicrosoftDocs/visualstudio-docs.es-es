# <a name="c28351"></a>C28351
Advertencia C28351: la anotación \<anotación > describe dónde no se puede usar un valor dinámico (una variable) en la condición.

 Esta advertencia se muestra cuando se aplica una anotación que se va a inspeccionar el tipo de C o C++.

 La expresión en el `_When_` se debe evaluar como una constante. El `_When_` se omite.
