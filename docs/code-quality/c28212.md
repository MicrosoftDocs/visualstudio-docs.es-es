# <a name="c28212"></a>C28212
Advertencia C28212: una expresión de puntero para la anotación

 Esta advertencia indica que el parámetro numerado a una anotación (no la función que se va a anotar) se espera que sea un puntero o tipo de matriz, pero se encontró algún otro tipo. La anotación debe corregirse.
