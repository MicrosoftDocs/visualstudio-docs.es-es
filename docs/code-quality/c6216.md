# <a name="c6216"></a>C6216
Advertencia C6216: conversión insertada por el compilador entre tipos enteros semánticamente diferentes: un tipo booleano HRESULT

 Esta advertencia indica que un valor booleano se usa como un `HRESULT` sin que se va a convertir de forma explícita. Los tipos booleanos indican una operación correcta mediante un valor distinto de cero; la operación correcta (`S_OK`) en `HRESULT` se indica mediante un valor de 0. El valor de error típicos para las funciones que devuelven un valor booleano false es un estado correcto cuando se prueba como un `HRESULT`. Esto es probable que conduzcan a resultados incorrectos.

## <a name="example"></a>Ejemplo
 El código siguiente genera esta advertencia:

```
#include <windows.h>
BOOL IsEqual(REFGUID, REFGUID);

HRESULT f( REFGUID riid1, REFGUID riid2 )
{
  // code ...
  return IsEqual(riid1, riid2);
}
```

 Para corregir esta advertencia, utilice el código siguiente:

```
#include <windows.h>
BOOL IsEqual(REFGUID, REFGUID);

HRESULT f( REFGUID riid1, REFGUID riid2 )
{
  if (IsEqual(riid1, riid2) == TRUE)
  {
    // code ...
    return S_OK;
  }
  else
  {
    // code ...
    return E_FAIL;
  }
}
```

 Para esta advertencia, el tipo `SCODE` es equivalente a `HRESULT`.

 Para obtener más información, consulte [SUCCEEDED Macro](http://go.microsoft.com/fwlink/?LinkId=92738) y [FAILED Macro](http://go.microsoft.com/fwlink/?LinkId=180875).
