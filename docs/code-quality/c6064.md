# <a name="c6064"></a>C6064
Advertencia C6064: falta el argumento de entero para \<función > corresponde al especificador de conversión \<número >  
  
 Esta advertencia indica que se proporcionaron no hay suficientes argumentos para que coincida con una cadena de formato y uno de los argumentos que faltan es un entero. Este defecto puede provocar resultados incorrectos.  
  
## <a name="example"></a>Ejemplo  
 El código siguiente genera esta advertencia porque se utiliza un número incorrecto de argumentos en la llamada a `sprintf` y el argumento que falta era un entero:  
  
```  
#include <string.h>  
void f( )  
{  
  char buff[15];  
  char *string="Hello, World";  
  
  sprintf(buff,"%s %d", string);  
}  
```  
  
 Para corregir esta advertencia, especifique argumentos que faltan, tal como se muestra en el código siguiente:  
  
```  
#include <string.h>  
void f( )  
{  
  char buff[15];  
  char *string = "Hello, World";  
  
  sprintf(buff,"%s %d",string, strlen(string));  
}  
```  
  
 El código siguiente utiliza la función de manipulación de cadena segura, `sprintf_s` para corregir esta advertencia:  
  
```  
#include <string.h>  
void f( )  
{  
  char buff[15];  
  char *string="Hello World";  
  
  sprintf_s(buff,sizeof(buff),"%s %d", string, strlen(string));  
}  
```
