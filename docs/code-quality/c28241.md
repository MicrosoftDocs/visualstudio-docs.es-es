# <a name="c28241"></a>C28241
Advertencia C28241: no se reconoce la anotación para la función del parámetro

 Se utilizó un nombre de la anotación no reconocido. Esto no debería ser posible si se utilizan las macros estándares para las anotaciones; Esta advertencia no se espera para ser vistas en uso típico.
