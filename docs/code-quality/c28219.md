# <a name="c28219"></a>C28219
Advertencia C28219: esperaba un miembro de enumeración para la anotación del parámetro en la anotación

 Se espera un parámetro a una anotación a ser un miembro de la instancia con nombre `enum` se encontró el tipo y algunos otros símbolos; utilice un miembro de ese `enum` tipo. Esto normalmente indica que una macro de anotación de código incorrecto.
