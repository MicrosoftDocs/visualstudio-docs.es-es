# <a name="search-tools-for-spy-views"></a>Herramientas de búsqueda para las vistas de Spy++
Utilice estos cuadros de diálogo para especificar qué datos se mostrará una ventana de vistas de Spy ++:  
  
## <a name="in-this-section"></a>En esta sección  
 [Cuadro de diálogo Buscar ventana](../debugger/find-window-dialog-box.md)  
 Se utiliza para ver las propiedades o mensajes de una ventana concreta.  
  
 [Cuadro de diálogo Búsqueda de ventana](../debugger/window-search-dialog-box.md)  
 Se usa para encontrar el nodo de una ventana concreta en la vista ventanas.  
  
 [Cuadro de diálogo Buscar mensaje](../debugger/message-search-dialog-box.md)  
 Se usa para encontrar el nodo de un mensaje concreto en la vista mensajes.  
  
 [Cuadro de diálogo Buscar proceso](../debugger/process-search-dialog-box.md)  
 Se usa para encontrar el nodo de un proceso específico en la vista de proceso.  
  
 [Cuadro de diálogo Buscar subproceso](../debugger/thread-search-dialog-box.md)  
 Se usa para encontrar el nodo de un subproceso concreto en la vista de subprocesos.
