# <a name="spy-views"></a>Vistas de Spy++
Spy ++ muestra las vistas de árbol de la windows, los mensajes, los procesos y subprocesos disponibles en el equipo.

## <a name="in-this-section"></a>En esta sección
 [Windows Vista](../debugger/windows-view.md) muestra una vista de árbol de los controles y ventanas disponibles.

 [La vista mensajes](../debugger/messages-view.md) muestra la secuencia de mensaje asociada a una ventana, proceso o subproceso.

 [Vista procesos](../debugger/processes-view.md) muestra una vista de árbol de procesos activos.

 [Vista de subprocesos](../debugger/threads-view.md) muestra una vista de árbol de subprocesos activos.
