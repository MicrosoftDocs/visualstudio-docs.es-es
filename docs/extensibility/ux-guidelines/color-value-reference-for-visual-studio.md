# <a name="color-value-reference-for-visual-studio"></a>Referencia de valores de color para Visual Studio

Este tema es una referencia para crear nuevos tokens de color. Al crear una nueva interfaz de usuario, puede usar colores similares de la interfaz de usuario existente para elegir los colores de cada tema principal y el nombre de color del sistema para ese elemento para contraste alto temas.

Para obtener información de implementación sobre cómo crear nuevos tokens de color, vea [Herramientas de tema de color.](../../extensibility/internals/color-theming-tools.md)

Para ver los tokens de color que se usarán con la interfaz de usuario existente, consulte [Colores compartidos Visual Studio](../../extensibility/ux-guidelines/shared-colors-for-visual-studio.md).

Al usar esta guía de referencia, recuerde:

- **Copie siempre los matizs de primer plano y de fondo.** No elija un primer plano de un elemento de la interfaz de usuario y un fondo de otro solo porque le gusten los matiz.

- **Se trata de códigos hexadecimales, no tokens de color.** No puede usar los nombres de token de color reales de estos elementos de la interfaz de usuario porque no puede controlar cómo podrían cambiarse los tokens en el futuro. En su lugar, cree tokens dentro de su propia categoría y use los valores de matiz que se muestran en las tablas. Los tokens de color para la interfaz de usuario existente se pueden encontrar [en Colores compartidos Visual Studio](../../extensibility/ux-guidelines/shared-colors-for-visual-studio.md).

- **contraste alto las muestras se muestran solo para la comparación.** Use el nombre de color del sistema contraste alto interfaz de usuario y no use un selector de colores ni cuentagotas en estas muestras. contraste alto muestra de este tema reflejan el contraste alto #1 en Windows. Los colores tendrán un aspecto diferente en otros contraste alto temas.

- **Para obtener mejores resultados, emparere los conjuntos de colores con la interfaz de usuario similar al ejemplo.**
