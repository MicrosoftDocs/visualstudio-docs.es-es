# <a name="color-value-reference-for-visual-studio"></a>Referencia del valor de color para Visual Studio

Este tema es una referencia para la creación de nuevos tokens de color. Al crear una nueva interfaz de usuario, puede usar colores similares a los de la interfaz de usuario existente para elegir los colores de cada tema principal y el nombre de color del sistema de ese elemento para contraste alto temas.

Para obtener información de implementación acerca de la creación de nuevos tokens de color, consulte Herramientas para el color de los [mismos](../../extensibility/internals/color-theming-tools.md).

Para ver los tokens de color que se van a usar con la interfaz de usuario existente, consulte [colores compartidos para Visual Studio](../../extensibility/ux-guidelines/shared-colors-for-visual-studio.md).

Al usar esta guía de referencia, recuerde:

- **Copie siempre los tonos de primer y segundo plano juntos.** No elija un primer plano de un elemento de la interfaz de usuario y un fondo desde otro solo porque le gustan los matices.

- **Estos son los códigos hexadecimales, no los tokens de color.** No puede usar los nombres de tokens de color reales de estos elementos de la interfaz de usuario porque no puede controlar cómo se pueden cambiar los tokens en el futuro. En su lugar, cree tokens dentro de su propia categoría y use los valores de matiz que se muestran en las tablas. Los tokens de color para la interfaz de usuario existente se pueden encontrar en [colores compartidos para Visual Studio](../../extensibility/ux-guidelines/shared-colors-for-visual-studio.md).

- **Contraste alto muestras solo se muestran para la comparación.** Use el nombre de color del sistema para la interfaz de usuario de contraste alto y no use un cuentagotas o selector de colores en estas muestras. Contraste alto muestras de este tema reflejan el contraste alto #1 tema de Windows. Los colores tendrán un aspecto diferente en otros temas de contraste alto.

- **Para obtener los mejores resultados, establezca pares de colores con la interfaz de usuario similar al ejemplo.**
