# <a name="list-of-available-services"></a>Lista de servicios disponibles

[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] y el SDK Visual Studio admiten los siguientes servicios. Algunos paquetes ofrecen sus propios servicios que no se enumeran aquí; por ejemplo, los servicios de lenguaje no tienen un único GUID de servicio. Debe usar el nombre del idioma para buscar el GUID del servicio de lenguaje en el Registro.

Use los GUID de servicio enumerados aquí u obtenidos de otro origen (por ejemplo, servicios de lenguaje) para obtener la interfaz principal o las interfaces que se muestran con cada servicio.
