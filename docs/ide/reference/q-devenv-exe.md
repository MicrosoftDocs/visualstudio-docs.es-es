# <a name="-devenvexe"></a>/? (devenv.exe)

Muestra un cuadro de mensaje con una lista de todos los modificadores `devenv` y una descripción breve de cada modificador.

## <a name="syntax"></a>Sintaxis

```shell
devenv /?
```
