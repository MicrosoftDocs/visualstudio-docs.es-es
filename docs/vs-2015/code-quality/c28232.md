# <a name="c28232"></a>C28232
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Advertencia C28232: \_Pre\_, \_Post\_, o \_Deref\_ no se aplican a ninguna anotación  
  
 Esta advertencia indica que un `_Pre_`, `_Post_`, o `_Deref_` operador aparece en una expresión de anotación sin una anotación funcional subsiguientes; se omitió el modificador, pero esto indica una anotación de código incorrecto.
