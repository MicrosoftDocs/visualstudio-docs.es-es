# <a name="c28230"></a>C28230
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Advertencia C28230: el tipo de parámetro no tiene ningún miembro.  
  
 Esta advertencia indica que un argumento a una anotación intenta obtener acceso a un `struct`, `class`, o `union` y el miembro con nombre no existe.
