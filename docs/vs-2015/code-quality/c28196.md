# <a name="c28196"></a>C28196
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Advertencia C28196: no se cumple el requisito. (La expresión no se evalúa en true.)  
  
 Esta advertencia indica que no tiene el prototipo de función para la función que se va a analizar un `__notnull`, `__null` o `__drv_valueIs` en un `_Out_` parámetro o el valor devuelto, pero el valor devuelto no es coherente con la anotación.
