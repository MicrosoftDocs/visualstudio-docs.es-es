# <a name="c28225"></a>C28225
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Advertencia C28225: no se encontró el número correcto de parámetros necesarios en la anotación  
  
 Esta advertencia indica que se utiliza la anotación con nombre con el número de parámetros incorrecto. Esto no debería ser posible si se utilizan las macros estándares para anotaciones. Esta advertencia no se espera que se muestra un uso típico.
