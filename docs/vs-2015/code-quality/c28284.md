# <a name="c28284"></a>C28284
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Advertencia C28284: para el símbolo, los predicados no se admiten para los símbolos que no son de función  
  
 Esta advertencia indica que una anotación condicional (predicado, `__drv_when`) no se encontró en un valor distinto de una función.
