# <a name="ienumdebugexpressioncontexts-interface"></a>IEnumDebugExpressionContexts (Interfaz)
Enumera una colección de `IDebugExpressionContexts` objetos.  
  
 Además de los métodos heredados de `IUnknown`, el `IEnumDebugExpressionContexts` interfaz expone los métodos siguientes.  
  
## <a name="methods-in-vtable-order"></a>Métodos en orden de Vtable  
  
|Método|Descripción|  
|------------|-----------------|  
|[IEnumDebugExpressionContexts::Next](../../winscript/reference/ienumdebugexpressioncontexts-next.md)|Recupera un número especificado de segmentos de la secuencia de enumeración.|  
|[IEnumDebugExpressionContexts::Skip](../../winscript/reference/ienumdebugexpressioncontexts-skip.md)|Omite un número especificado de segmentos en una secuencia de enumeración.|  
|[IEnumDebugExpressionContexts::Reset](../../winscript/reference/ienumdebugexpressioncontexts-reset.md)|Restablece una secuencia de enumeración al principio.|  
|[IEnumDebugExpressionContexts::Clone](../../winscript/reference/ienumdebugexpressioncontexts-clone.md)|Crea un enumerador que contiene el mismo estado que el enumerador actual.|
