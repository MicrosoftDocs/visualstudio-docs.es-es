# <a name="illegal-assignment-javascript"></a>Asignación no válida (JavaScript)
Se intentó asignar un valor a un identificador de sólo lectura. No se puede asignar un valor a un identificador de sólo lectura. Por ejemplo, el host define objetos y objetos COM externos.  
  
### <a name="to-correct-this-error"></a>Para corregir este error  
  
-   No asigne valores a identificadores de sólo lectura.
