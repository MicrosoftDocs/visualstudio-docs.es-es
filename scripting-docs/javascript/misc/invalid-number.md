Número no válido (ErrbadNumber 1013)  
  
 Ha intentado usar un número que no es válido en JavaScript.  
  
 Para obtener información sobre los números válidos, vea [Number (objeto](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Number)).
